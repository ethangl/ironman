# ironman.fm Roadmap

## Gameplay
- [x] Challenges — challenge a friend to an ironman duel on a specific song. First to surrender loses.
- [x] Milestones — badges at 10, 25, 50, 100 plays. "Century Club" for 100+.
- [x] Hall of Fame — personal profile page showing your streak history.

## Social
- [x] Live feed — real-time global feed of lock-ins, surrenders, milestones.
- [x] Song pages — `/song/[trackId]` with stats: total plays, attempts, avg streak, current Iron Man.

## Spicy
- [x] Shame board — fastest surrenders.
- [x] Weakness stats — show weakness count on leaderboard entries.
- [x] Hardcore mode — pausing or quitting breaks your streak entirely.

## Polish
- [x] Progress bar — show current play progress toward completion.
